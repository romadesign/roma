El taller está dirigido a empresarios, emprendedores y público en general con CERO CONOCIMIENTO DE SISTEMAS OPERATIVOS COMO WINDOWS. El curso va paso a paso desarrollado con una adecuada metodología y enseñanza personalizada.

Descripción del curso

En este curso se explicara de manera clara y paso a paso todo lo que hay que tener en cuenta antes, durante y después de formatear una computadora, incluyendo ejemplos prácticos de todos los temas vistos.

Temario del curso:

  • 1 Que es formatear, para que sirve y tipos de formateo 02:00 hrs
    1.1 Ingreso. Que es formatear 00:15 minutos
    1.2 Para qué sirve el formateo 00:15 minutos
    1.3 Elegir Sistema Operativo a instalar 00:15 minutos
    1.4 Analizar todas las opciones 00:15 minutos
    1.5 Elegir Windows de 32 bits (x86) o 64 bits (x64) 00:20 minutos
    1.6 Drivers 00:20 minutos
    1.7 Elegir drivers adecuados para la computadora y el nuevo sistema operativo que se va a instalar 00:20 minutos
  • 2. Paso II. 02:00 hrs
    2.1 Respaldar información 00:15 minutos
    2.2 Particiones del disco duro 00:15 minutos
    2.3 Para qué sirven las particiones 00:15 minutos
    2.4 Creación de Particiones 00:15 minutos
    2.5 Crear un Pendrive Booteable 00:20 minutos
    2.6 Bootear Pendrive con Windows 00:20 minutos
    2.7 Realizar la instalación y activar drivers 00:20 minutos

Requerimiento para el curso

Un ordenador gama medía Intel Core I3

Con una memoria RAM minima de 4GB

Un Pendrive (USB) 8GB
